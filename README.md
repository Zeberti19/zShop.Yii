# zShop.Yii
Тестовый магазин на базе Yii фреймворка для тестирования веб-технологий.

После выкачивания проекта, помимо стандартных настроек, связанных с Yii2 фреймворком, нужно выполнить еще одну настройку, чтобы система работала без ошибок.
Система предполгает возможность обращения к различным её разделам по URL, в котором есть закодированные слеши. Поэтому вы должны убедиться, что ваш веб-сервер разрешает обработку URL, где закодирован слеш. Если вы используете Apache, то укажите следующую настройку для вашего хоста:
AllowEncodedSlashes NoDecode
